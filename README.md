# florir

Projeto front-end React + Vite para a aplicação `florir`.

## Tecnologias usadas

- React 18
- TypeScript
- Vite
- Tailwind CSS v4 (`@tailwindcss/vite`)
- `tw-animate-css`
- pnpm (workspace)

## Como rodar localmente

1. Instale as dependências:

```bash
pnpm install
```

2. Inicie o modo de desenvolvimento:

```bash
pnpm dev
```

3. Abra o navegador em `http://localhost:5173`

## Build de produção

```bash
pnpm build
```

## Observações

- O arquivo `src/styles/tailwind.css` já está configurado para carregar Tailwind.
- O `.gitignore` já ignora dependências e arquivos de build.

## Sobre o projeto

Este repositório contém a interface principal da aplicação, com rotas, componentes e o tema do dashboard.
